Title: scanner_api.sh

Purpose: Main script to run a scanner

Instruction:

1. Open 'Terminal'

2. Set current directory that contains scripts
i.e. if 'Scripts' forlder contains scripts type line between " " in the command-line 
"cd /Desktop/Scripts"

3. Find out required information as follows
a: scanner ID that you want to run
i.e. 4, single digit number that attached to each scanner
b: configuration text file that you want to run with
i.e. config.txt

4. Type following line in the command-line to run scanner 
Replace [a] and [b] with specific information from 3
"./scanner_api.sh [a] [b] &"

5. Repeat 1 to 4 to run anther scanner. (Do not run same scanner again.)
